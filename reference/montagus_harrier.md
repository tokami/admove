# Montagu's harrier

A simulated data set of mark-resight tags for Montagu's harrier

## Usage

``` r
montagus_harrier
```

## Format

The complete simulated data set of class `admove_sim` containing:

- grid:

  The simulation grid of class `admove_grid`

- cov:

  The covariance fields of class `admove_cov`

- par_sim:

  The "true" parameters used for simulation

- tags:

  Simulated tags (here: make-resight tags)

- dat:

  The `admove` input data of class `admove_data`

- conf:

  A list with configuration settings

- par:

  A list with initial parameters

- map:

  A list with the mapping of parameters
