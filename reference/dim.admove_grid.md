# Return dimensions

Return dimensions of a grid of class `admove_grid`.

## Usage

``` r
# S3 method for class 'admove_grid'
dim(x)
```

## Arguments

- x:

  Grid of class `admove_grid` as returned by the function
  [`create_grid()`](https://tokami.github.io/admove/reference/create_grid.md).

## Value

A vector with x and y dimensions (number of cells).
