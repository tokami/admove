# Print admove objects

Print methods for `admove` classes.

## Usage

``` r
# S3 method for class 'admove_cov'
print(x, ...)

# S3 method for class 'admove_data'
print(x, ...)

# S3 method for class 'admove'
print(x, ...)

# S3 method for class 'admove_grid'
print(x, ...)

# S3 method for class 'admove_tags'
print(x, ...)

# S3 method for class 'admove_tags'
print(x, ...)
```

## Arguments

- x:

  An object to print.

- ...:

  Further arguments passed to or from other methods.

## Value

The object is printed to the console. Invisibly returns `x`.
