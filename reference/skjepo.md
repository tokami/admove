# Skipjack tuna in the Eastern Pacific Ocean

A simulated data set of data-storage and mark-recapture tags for
skipjack tuna in the Eastern Pacific Ocean

## Usage

``` r
skjepo
```

## Format

A list with:

- sim:

  The complete simulated data set of class `admove_sim`

- grid:

  The simulation grid of class `admove_grid`

- cov:

  The covariance fields of class `admove_cov`

- ctags:

  Simulated mark-recapture tags

- dtags:

  Simulated data-storage tags

- fit:

  Fitted admove object of class `admove`
