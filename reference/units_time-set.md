# Set temporal units

Replacement function for
[`units_time()`](https://tokami.github.io/admove/reference/units_time.md).

## Usage

``` r
units_time(x) <- value

# Default S3 method
units_time(x) <- value

# S3 method for class 'admove_tref'
units_time(x) <- value

# S3 method for class 'admove_cov'
units_time(x) <- value

# S3 method for class 'admove_tags'
units_time(x) <- value

# S3 method for class 'admove_data'
units_time(x) <- value
```

## Arguments

- x:

  An object to modify.

- value:

  A character string describing the temporal units.

## Value

`x` with updated temporal units.
