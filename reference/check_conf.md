# Check and complete a model configuration list

Checks whether a configuration list contains all required settings for
`admove`. Missing settings are filled in from
[`default_conf()`](https://tokami.github.io/admove/reference/default_conf.md).

## Usage

``` r
check_conf(conf = NULL, dat, verbose = TRUE)
```

## Arguments

- conf:

  An optional configuration list. If `NULL`, the default configuration
  generated by
  [`default_conf()`](https://tokami.github.io/admove/reference/default_conf.md)
  is returned.

- dat:

  A data list containing model input data, as produced by
  [`setup_data()`](https://tokami.github.io/admove/reference/setup_data.md).

- verbose:

  Logical; if `TRUE`, informative messages are printed when missing
  settings are added or when default settings are used.

## Value

A named configuration list containing all required settings.

## Details

The function compares the user-supplied configuration list with the
default configuration produced by
[`default_conf()`](https://tokami.github.io/admove/reference/default_conf.md).
Any missing named entries are added, while existing entries in `conf`
are preserved unchanged.

This function does not validate whether supplied configuration values
are internally consistent; it only ensures that all required settings
are present.

## Examples

``` r
conf <- list(use_taxis = FALSE)
conf <- check_conf(conf, skjepo$dat)
#> Added missing configuration setting(s): use_ctags, use_dtags, use_stags, use_advection, obs_var_type, do_update, engine, ctmc_method, seasonal_cov, seasonal_spline
```
