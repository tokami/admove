# Set spatial units

Replacement function for
[`units_space()`](https://tokami.github.io/admove/reference/units_space.md).

## Usage

``` r
units_space(x) <- value

# Default S3 method
units_space(x) <- value

# S3 method for class 'admove_sref'
units_space(x) <- value

# S3 method for class 'admove_grid'
units_space(x) <- value

# S3 method for class 'admove_cov'
units_space(x) <- value

# S3 method for class 'admove_tags'
units_space(x) <- value

# S3 method for class 'admove_data'
units_space(x) <- value
```

## Arguments

- x:

  An object to modify.

- value:

  A character string describing the spatial units.

## Value

`x` with updated spatial units.
