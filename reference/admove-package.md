# admove: Estimating animal movement with advection–diffusion models

A movement model that allows estimation of habitat preferences and
fine-scale movement rates based on advection-diffusion models using
various types of tagging data. It supports flexible spatiotemporal
modeling with options for Kalman filter or matrix exponential
approaches, and includes tools for data preparation, model fitting, and
visualisation.
