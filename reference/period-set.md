# Set temporal period

Replacement function for
[`period()`](https://tokami.github.io/admove/reference/period.md).

## Usage

``` r
period(x) <- value

# Default S3 method
period(x) <- value

# S3 method for class 'admove_tref'
period(x) <- value

# S3 method for class 'admove_cov'
period(x) <- value

# S3 method for class 'admove_tags'
period(x) <- value

# S3 method for class 'admove_data'
period(x) <- value
```

## Arguments

- x:

  An object to modify.

- value:

  A period value.

## Value

`x` with updated period information.
