# Description of model

Writes a string to install the version of the package which was used to
run the model.

## Usage

``` r
model_version_info(fit, ...)
```

## Arguments

- fit:

  A fitted admove list of class `admove` as returned by the function
  [admove](https://tokami.github.io/admove/reference/admove.md).

- ...:

  Additional parameters to be passed to
  [message](https://rdrr.io/r/base/message.html).
